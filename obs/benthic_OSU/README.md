# README for benthic_OSU

#### This is an archive of biogeochemical parameters collected from bottom boundary layer off Oregon shelf waters by PI Clare E. Reimers (Oregon State University).

---


- Two datasets were available at BCO-DMO. 

- The bottle data is publicly available at: https://www.bco-dmo.org/dataset/793115
- The time-series dataset "aquatic eddy covariance measurements" is publicly available at https://www.bco-dmo.org/dataset/962251

- You can check her papers
https://aslopubs.onlinelibrary.wiley.com/doi/full/10.1002/lno.70197 and https://www.sciencedirect.com/science/article/abs/pii/S0304420324001245
